## 🛠️ TECHNOLOGY STACK

### Backend
- **Runtime**: Node.js 18
- **Framework**: Express.js
- **Database**: MongoDB (Cosmos DB for Azure)
- **Cache**: Redis
- **NLP**: Natural.js + Rasa
- **Auth**: JWT tokens
- **Real-time**: WebSocket

### Frontend
- **Framework**: React 18
- **State**: Zustand
- **Styling**: CSS + Tailwind
- **HTTP**: Axios
- **Charts**: Recharts

### DevOps
- **Containerization**: Docker & Compose
- **Cloud**: Microsoft Azure
- **CI/CD**: GitHub Actions ready
